# Different literals

## Decimal Integer literals

- 0 to 9
- must start with a number other than 0
- no decimal places or commas

## Hexadecimal Integer literals

- must begin wth 0 (ex: 0x)
- Digits 0 to 9 and A to F / a to f

## Binary Integer Literals

- Must beging with 0b or 0B
- May include digits 0 and 1

## Floating Point Literals

- like decimal integer literals except the decimal point is allowed
- 'e' notation is used to specify exponents
